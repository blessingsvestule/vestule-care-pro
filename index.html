📅 Add to Google Calendar

Tap each upcoming appointment to add it to your Google Calendar. Set as Alarm for sound alerts.

⚠️ Set sound alerts

Android: Open the event → Add notification → change to Alarm → set 1 week before AND 1 day before → Save. Then Settings → Apps → Google Calendar → Notifications → High Importance.

iPhone: Open the event → Add alert → 1 week before AND 1 day before → Save. Settings → Sounds → Calendar Alerts → select a ringtone.
